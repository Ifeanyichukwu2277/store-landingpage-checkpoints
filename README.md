
# ORIGIN - Activewear-Landing-Page

A clean Activewear store landing page designed with HTML and CSS


# ORIGIN

“ORIGIN” was chosen as the brand name to represent authenticity, identity, and the starting point of style
 and performance. The name reflects the idea of returning to the basics of comfort, movement, and confidence in
 activewear. It also creates a strong, memorable brand identity that aligns with the minimalist and modern design 
 of the landing page.

# Typography & Readability:

Chose Poppins, a modern sans-serif font, across the body and headings for clean and consistent readability.
Font sizes are scaled for desktop and mobile to maintain hierarchy and clarity.


# Navbar Design & Layout:

Navbar items are displayed inline with spacing for desktop and stacked vertically on smaller screens (max-width: 768px)
 for responsiveness. Flexbox ensures the navbar spreads across the header, and wrapping allows items to adapt when the 
 screen shrinks.

# Hero Section:

Used a linear gradient background for a visually appealing hero area.
Text and buttons are centered, with flexible padding and width for different screen sizes.
On mobile (max-width: 576px), the hero text and image stack vertically for a better mobile layout.


# Buttons:

Buttons have rounded corners, distinct background colors, and hover effects for clarity and interaction.
Width is kept natural on small screens to avoid stretching across the page.


# Product Cards:

Cards are arranged in a flex container with space between them for desktop.
On tablets (max-width: 992px), cards adjust to 2 per row, and on mobile (max-width: 576px), they stack 1 per row.
The Origin tag is positioned absolutely over images for brand visibility.
Product colors are displayed as small clickable dots, with visible borders for contrast.


# Footer Design:

Footer uses flexbox, distributing links and social icons evenly on desktop.
On smaller screens, items stack vertically and center-align, ensuring readability and accessibility.
Quick links and social icons maintain spacing and hover effects for a clean, interactive footer.


# Overall Responsiveness:
# Media queries 

(max-width: 992px, 768px, 576px) adjust layout, text size, card width, and navbar stacking.
The design prioritizes visual clarity, easy navigation, and adaptability across devices, ensuring a professional
 and user-friendly interface.

 # Why An Activwear Store

 An activewear landing page was chosen because it provides a clear opportunity to showcase visual products, branding,
 and user interaction. Activewear relies heavily on imagery, color variations, and product presentation, making it ideal 
 for practicing layout design, card structures, and responsive styling. Additionally, it allows for the integration of modern
 UI elements such as hero sections, call-to-action buttons, and product grids, which are essential skills in front-end development.
